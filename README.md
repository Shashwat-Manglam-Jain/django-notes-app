# Django Notes App

A role-based notes management system built with **Django + DRF (JWT auth)** and **Next.js (App Router + Tailwind)**.

---

## Tech Stack

### Backend
- Django
- Django REST Framework
- SimpleJWT (Authentication)
- SQLite (default)

### Frontend
- Next.js (App Router)
- TypeScript
- Tailwind CSS

---

## Prerequisites

Make sure you have the following installed:

- Python **3.10+**
- Node.js **18+**
- npm or yarn
- Git
